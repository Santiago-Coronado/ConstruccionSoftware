Pasos para Clonar un Repositorio y Subir Cambios

Esto ya toma en cuenta que tienes cuenta de Github y Git instalado

  1. Paso 1

    1. Hacer el comando git clone en la direccion de la computadora correspondiente

  2. Paso 2

    1. Hacer el comando cd [nombre de repositorio] para entrar a la direccion correspondiente

  3. Paso 3

    1. Hacer los cambios correspondiente correspondiente

  4. Paso 4

    1. Hacer el comando git add [nombre de archivo] para poner los cambios en el staging area

  5. Paso 5

    1. Hacer el comando git commit -m "[mensaje]" para confirmar cambios y dejar un mensaje o descripcion del cambio

  6. Paso 6

    1. Hacer el comando git push -u origin [rama] para subir los cambios a Github
