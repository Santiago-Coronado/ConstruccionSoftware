Comandos

Comando Descripcion
cd [camino url] Va a esa dirrecion en la computadora
cd ~ Te regresa a tu punto de origen en la computadora
ls Enseña los nombres de lo que hay en ese directorio
ls -al Enseña todo los detalles o datos de las cosas en ese directorio
clear Limpia terminal
mkdir [nombre de archivo] Crea carpeta con ese nombre
touch [nombre de archivo] Crea archivo con ese nombre
tab Autocompleta
cat Concatena archivo
nano/vi Edita archivo
cp [que copias][a donde] Hace copia y la pone en la direccion deseada
mv [que mueves][a donde] Mueve archivo a esa direccion
rm [nombre de archivo] Remueve el archivo
rm -r [nombre de carpeta] Remueve carpeta y sus contenidos
git clone [llave ssh] Clona repositorio en la computadora
git add [nombre de archivo] Agrega archivo a staging area
git commit -m "[mensaje]" Confirma los cambios y le pone un mensaje o descripcion
git push -u origin [rama] Sube cambios a github desde la rama correspondiente
pwd Muestra ruta actual en la que estas
